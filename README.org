* About "dcpu-el"

  dcpu-el is DCPU-16 written for emacs.

* Quickstart

  git clone .... dcpu-el.git
  cd dcpu-el.git
  emacs test-hello-emacs.dasm
  eval

* Features
  emulator with register, memory and breakpoints
  assembler
