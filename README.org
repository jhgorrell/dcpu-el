* About "dcpu-el"
  dcpu-el is a DCPU-16 assembly mode and emulator written in elisp.

* Quickstart

:  git clone https://github.com/jhgorrell/dcpu-el.git dcpu-el.git
:  cd dcpu-el.git
:  emacs test-hello-emacs.dasm

* Features
  - emulator with register, memory and breakpoints
  - assembler
